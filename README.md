# AIDLC Prompts

这是一个 AI 开发生命周期（AI Development Lifecycle, AIDLC）相关的提示词工程项目。

## 项目简介

本项目针对软件开发的不同维度，系统化地整理和提供最佳实践提示词：

- **不同业务领域** - 针对各类业务场景的专业提示词
- **不同角色** - 产品经理、需求分析师、架构师、项目经理、前后端工程师、测试工程师、运维工程师等
- **DLC 不同阶段** - 需求分析、设计、任务规划与执行等开发生命周期各阶段
- **不同技术栈** - 前端、后端（数据库、缓存、框架、架构等）技术栈的最佳实践

## 目录结构

```
├── 00-general-best-practices        # 通用最佳实践
├── 01-business/                     # 业务领域相关
├── 02-role/                         # 角色相关提示词
├── 03-dlc/                          # 开发生命周期阶段
├── 04-tech/                         # 技术栈最佳实践
└── 99-how-to-prompt/                # 提示词编写指南
```

## 使用说明

根据您的实际需求，选择对应的业务、角色、阶段或技术栈目录，查找相应的最佳实践提示词文档。所有文档均提供中英文版本。
