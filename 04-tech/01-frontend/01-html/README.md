# HTML 开发提示词

## 角色设定

你是一位精通 HTML5 的前端开发专家，具有丰富的语义化标签使用经验和无障碍访问知识。

## 核心能力

- HTML5 语义化标签的正确使用
- 表单设计与验证
- 无障碍访问 (ARIA) 实现
- SEO 优化的 HTML 结构
- 多媒体元素的嵌入与控制

## 提示词模板

### 语义化结构

```
请帮我创建一个语义化的 HTML 页面结构，包含：
- 页面类型：[博客文章/产品页/登录页/...]
- 主要区块：[头部导航/侧边栏/主内容区/页脚]
- SEO 要求：[是/否]
- 无障碍要求级别：[A/AA/AAA]

请确保：
1. 使用正确的 HTML5 语义标签
2. 包含必要的 meta 标签
3. 添加适当的 ARIA 属性
4. 结构层次清晰
```

### 表单设计


## 最佳实践

1. **始终使用 DOCTYPE 声明**：`<!DOCTYPE html>`
2. **设置正确的语言属性**：`<html lang="zh-CN">`
3. **使用语义化标签替代 div**：header, nav, main, section, article, aside, footer
4. **为图片添加 alt 属性**：描述图片内容或留空（装饰性图片）
5. **使用 loading="lazy"**：延迟加载非首屏资源
6. **表单元素关联 label**：使用 for 属性或嵌套
7. **合理使用 heading 层级**：保持逻辑顺序，不跳级
8. **添加 meta viewport**：确保移动端响应式

## 常见问题检查清单

- [ ] 是否有且仅有一个 h1 标签？
- [ ] 所有图片是否都有 alt 属性？
- [ ] 表单元素是否都有关联的 label？
- [ ] 链接文本是否具有描述性？
- [ ] 是否避免使用表格进行布局？
- [ ] 是否设置了正确的 lang 属性？
- [ ] 是否包含必要的 meta 标签？
